Guide on how to run getToken.py

Download getToken.py and put it in a place in your computer that you know how to access through your 
command line.

Install python in your computer and make sure it is in the PATH


Run the necessary installs:
pip install requests
pip install azure-devops

then Run the program and follow instructions.
To run the program navigate to the directory where it is using the command line and use
python getToken.py

a web browser will open, accept the message and then copy the redirected URL into the command line.
